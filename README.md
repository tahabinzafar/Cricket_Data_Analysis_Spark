
# Cricket Data Analysis using PySpark

The Indian Premier League (IPL) is the biggest league in the cricketing world.

This repository aims to utilize PySpark, a powerful analytics tool, to conduct a comprehensive analysis of IPL data. The dataset required for this analysis is accessible via Amazon S3:

```bash
  s3://ipl-data-analysis-project/Ball_By_Ball.csv
```

Through Apache Spark's distributed computing capabilities, this project endeavors to uncover valuable insights into various aspects of IPL cricket, facilitating deeper understanding and informed decision-making within the realm of sports analytics.
